# Проект "Посмотри в окно"

Проектная работа по созданию интерактивного веб-приложения для просмотра видео из разных городов в разное время суток.

## Технологии
- HTML5
- CSS3 (Grid, Flexbox)
- JavaScript

## Функциональность
- Поиск видео по городу и времени суток
- Просмотр выбранного видео в основном блоке
- Подгрузка дополнительных карточек
- Адаптивный интерфейс

## Структура проекта
- `index.html` - основная разметка
- `style.css` - стили проекта
- `scripts/` - директория для JavaScript файлов
- `fonts/` - директория для шрифтов

## Запуск
Откройте файл `index.html` в любом современном браузере.
